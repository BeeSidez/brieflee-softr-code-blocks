Welcome email · 4 options for new signups

Each option is rendered inside a faux desktop-mail-client frame so you can read it like a real inbox message — same subject line, sender, preheader, and 600 px body column the actual email would use. Click any artboard to open it full-bleed.

  • E1 · Editorial Engraving — premium, magazine feel. Engraving illustration hero, signed personal note from a co-founder, single CTA. Good for "first impression matters."
  • E2 · Friendly Sticker — warmer, brighter. Rocket sticker hero, three numbered "get started" steps, primary + secondary CTA. Good for self-serve audiences.
  • E3 · Score Card Demo — leads with a fully-rendered sample score card so the user sees the product output the moment they open the mail. Reads as utility, not marketing.
  • E4 · Minimal Type-First — no imagery in the body. All-type hero, single CTA, three quick "while you're here" links. Linear / Stripe energy.

All four use the same brand foundation: Plus Jakarta Sans, navy headings, periwinkle accents, no em dashes, sentence case, no emoji in body copy (E2's subject uses one — easy to drop). Personalization tokens (Sarah) are placeholders.
