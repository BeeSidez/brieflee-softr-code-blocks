# Brieflee — Marketing UI Kit

Recreation of the brieflee.co marketing site, built with React + Tailwind-style classes powered by `colors_and_type.css`.

## Components
- `Nav.jsx` — top nav with wordmark, links, sign-in / try-free
- `Hero.jsx` — periwinkle gradient hero with eyebrow pill, headline, dual CTA, animated demo
- `LogoStrip.jsx` — "Trusted by" social proof
- `FeatureGrid.jsx` — three-up feature cards with sticker illustrations
- `FeatureRow.jsx` — alternating image + text rows using engravings
- `Stats.jsx` — three large numbers in a navy strip
- `CTA.jsx` — full-width gradient CTA with rocket sticker
- `Footer.jsx` — navy footer with wordmark and link columns

## index.html
Click "Try free" to advance the periwinkle pill ribbon (a tiny in-page interaction). Otherwise this is a scrolling marketing page.
